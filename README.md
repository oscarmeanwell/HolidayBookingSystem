# README #


### Assessment 2 2016 - SD2 ###

* Written in C#
* Uses WPF

### How do I get set up? ###

* Clone the repo
* Add the files found in the repo - cust.csv, guest.csv, tmp.csv, guestno.txt, refno.txt to a D drive
* If you dont have a D drive, do a find and replace on 'D:\\' to your target
* Run the project
